# VGhack

Как установить:
1) Скачайте файл со страницы релиза: https://github.com/su57ks/VGhack/releases
2) Распакуйте файл в папку
3) Откройте Google Chrome
4) Три точки -> Настройки -> Расширения
5) В правом верхнем углу включите Режим разработчика
6) Нажмите Загрузить распакованное
7) В появивщемся окне выберите папку, в которую Вы распаковали архив в пункте 2
8) Удостоверьтесь, что расширение активно (ползунок на карточке расширения)
9) Хорошего использования!

Как использовать:
1) Перейдите на сайт
2) Выберите нужное задание
3) Вверху появится всплывающее окно вида "ПРАВИЛЬНЫЙ ОТВЕТ: Господин Айзен"
4) Нажмите Ок и введите ответ
5) Все юудет повторяться с пункта 3
